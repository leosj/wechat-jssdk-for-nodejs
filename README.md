#wechat-jssdk-for-nodejs
##用nodejs书写微信sdk的示例

缓存数据库用的是redis，accesstoken以及jsapiticket都存放在这里面

##主要示例

*     <h4>一、图片类相关接口</h4>
            1、拍照或从手机相册中选图接口
			2、预览图片接口
			3、上传图片接口(演示上传选中最后一张图片)
			4、下载图片接口(演示下载刚刚上传的图片)
* <h4>二、界面类操作</h4>
    		1、显示/隐藏右上角菜单接口
			2、关闭当前网页窗口接口
			3、批量显示/隐藏功能按钮接口
			4、显示/隐藏所有非基础按钮接口
* <h4>三、音频类接口</h4>
    		1、开始/停止录音
			2、播放/暂停录制的录音
			3、停止播放录音
		    4、上传录音
			4、下载录音
			5、识别语音
* <h4>四、其他接口</h4>
    		1、查看网络制
			2、获取位置

##遇到的坑
*  微信demo示例中签名算法使用的是jssha，但是jssha在1.6版本以上就不支持

```javascript
shaObj = new jsSHA(string, 'TEXT');
```
*  这样的写法了
所以用的require('sha1')
